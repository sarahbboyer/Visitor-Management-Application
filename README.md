# Corporate Visitor Check-In Kiosk

A Vite + React app for corporate visitor intake, photo capture, badge printing, and host notification workflow.

## Features

- Visitor check-in flow with host routing details
- Camera capture for badge photos
- Screening status and activity feed
- Host notification payloads (email/SMS endpoint hook)
- Badge print layout
- Visitor log export to CSV
- Local persistence with `localStorage`

## Run (Development)

1. Install dependencies: `npm install`
2. Start Vite dev server: `npm run dev`
3. Open the local URL shown in terminal (typically `http://localhost:5173`)

## Build

1. `npm run build`
2. `npm run preview`
